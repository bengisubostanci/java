# Java Basics Practices

A simple console application featuring 3 basic algorithms in Java, demonstrating user input handling and fundamental conditional logic.

## 🚀 Features

1. **GPA Calculator:** Takes grades for 4 courses and calculates the average.
2. **VAT Calculator:** Adds 18% VAT to an entered amount and prints the total.
3. **Activity Suggester:** Recommends an activity based on the temperature (Skiing, Theatre, Barbecue, Swimming).

> ⚠️ **Note:** Code variables and console outputs are written in **mixed Turkish and English** for learning purposes.

## 💻 How to Run

```bash
javac Main.java
java Main
